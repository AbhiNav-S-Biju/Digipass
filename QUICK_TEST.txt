╔══════════════════════════════════════════════════════════════════════════════╗
║ TROUBLESHOOTING QUICK FIXES                                                  ║
╚══════════════════════════════════════════════════════════════════════════════╝

❌ Connection refused
   → npm start not run
   → Run in correct folder: c:\Users\Abhinav\Desktop\Digipass

❌ "Cannot find module"
   → npm install not run
   → Run: npm install

❌ "Email already registered"
   → Email exists in database
   → Use different email

❌ Unauthorized / No token
   → Missing Authorization header
   → Must add: Authorization: Bearer TOKEN

❌ Invalid token / Forbidden
   → Token copied wrong or expired
   → Get fresh token from /login endpoint

❌ Database connection error
   → PostgreSQL not running
   → Ensure "Digipass" database exists
   → Check credentials in db.js
